# **App Name**: Web Weaver Portfolio

## Core Features:

- Hero Section: Displays a welcoming introduction with the user's name and profession.
- About Section: Displays the user's background, skills, and experience.
- Work Section: Displays the user's portfolio of work with links to live projects and GitHub repositories.
- Contact Section: Displays contact information and a contact form for potential clients to get in touch.
- Resume Download: Allows visitors to download the user's resume in PDF format.
- Social Media Links: Links to the user's social media profiles such as LinkedIn, Twitter, and GitHub.

## Style Guidelines:

- Primary color: Dark navy blue (#1E3A8A) to convey professionalism and trust.
- Background color: Very dark navy blue (#0F172A) to provide a strong contrast for the content.
- Accent color: Teal (#2DD4BF) for highlights and interactive elements, providing a fresh and modern feel.
- Headline font: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel for headings and important text.
- Body font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look for body text.
- Use simple, line-based icons for navigation and social media links.
- A clean, minimalist layout with plenty of white space to make the content easy to read and scan.
- Subtle animations and transitions to add a touch of polish and engagement.
